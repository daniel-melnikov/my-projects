## Описание проекта
Необходимо подготовить данные для статьи о развитии индустрии видеоигр в начале XXI в., а именно:
- Проверить корректность данных;
- Провести предобработку данных;
- Получить срез данных с 2000 по 2013 гг.;

## Используемые библиотеки:
- pandas

## Результаты:
- Приведены к единому стилю названия полей в данных;
- Данные приведены к корректному формату;
- Произведена работа с пропусками: удаление, замена средним значением;
- Удалены явные дубликаты;
- Произведена фильрация данных по годам;
- Произведена категоризация данных по оценкам пользователей и критиков;
- Определены платформы, выпустившие наибольшее количество видеоигр в заданный период;
