# Анализ трендов на рынке видеоигр

Ссылки на вкладки дашборда: [Анализ игр](https://github.com/daniel-melnikov/my-projects/blob/main/Анализ%20трендов%20на%20рынке%20видеоигр/Анализ%20игр.png), [Анализ игроков](https://github.com/daniel-melnikov/my-projects/blob/main/Анализ%20трендов%20на%20рынке%20видеоигр/Анализ%20игроков.png), [Игровые достижения](https://github.com/daniel-melnikov/my-projects/blob/main/Анализ%20трендов%20на%20рынке%20видеоигр/Игровые%20достижения.png), [Стоимость игр](https://github.com/daniel-melnikov/my-projects/blob/main/Анализ%20трендов%20на%20рынке%20видеоигр/Стоимость%20игр.png), [Инструкция к дашборду](https://github.com/daniel-melnikov/my-projects/blob/main/Анализ%20трендов%20на%20рынке%20видеоигр/Инструкция%20к%20дашборду.png).

## Описание проекта:
Необходимо создать дашборд, который позволит отслеживать тренды на рынке видеоигр.

## Результаты:
- На вкладке "Анализ игр" рассматривается количество игровых релизов по платформам, их распределение по жанрам, календарь выхода игр, а также количество разработчиков и издателей;
- На вкладке "Анализ игроков" изучены поведенческие особенности пользователей;
- На вкладке "Стоимость игр" выявлены изменения стоимости в разрезе различных категориальных данных;
- На вкладке "Игровые достижения" обзорно представлено распределение достижений по категориальным данным;
- Вкладка "Инструкция к дашборду" содержит информацию о дате создания, ответственном лице, источниках данных и структуре дашборда;
- Добавлена фильтрация по периоду, гранулярности и категориальным данным, а также фильтр по месяцу для графика "Изменение количества игроков год к году по платформам.
