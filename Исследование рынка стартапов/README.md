# Исследование рынка стартапов

[Ссылка на проект](https://github.com/daniel-melnikov/my-projects/blob/main/Исследование%20рынка%20стартапов/Исследование%20рынка%20стартапов.ipynb)

## Описание проекта
**Цель проекта** - оценить перспективы выхода на рынок стартапов для заказчика.
**Задачи:**
- Провести предобработку данных;
- Классифицировать компании по срокам финансирования, сегменты рынка;
- Исключить аномальные объемы финансирования, сравнить типичные;
- Проанализировать динамику объемов инвестиций по годам;
- Оценить долю ROI;
- Дать рекомендации заказчику на основе сделанных выводовж

## Используемые библиотеки
- pandas
- matplotlib

## Результаты:
- Данные очищены, приведены к единому стилю и к корректным форматам;
- Пропуски заполнены срединной датой и заглушками;
- Удалены явные дубликаты;
- Данные категоризированы по срокам финансирования и по сегментам рынка;
- Определены и отфильтрованы выбросы;
- Проведен анализ типов финансирования по объему и популярности, динамики финансирования по годам;
- Рассмотрена динамика размера общего финансирования по массовым сегментам рынка, которые показали прирост в 2014 г.;
- Изучен ROI по годам и типам финансирования;
- Даны рекомендации по выбору отраслей и типов для инвестиций.
