## Описание проекта
Заказчик планирует выйти на рынок Санкт-Петербурга и Ленинградской области.  
**Задача:** Подготовить анализ объявлений о продажи жилой недвижимости в данных регионах, чтобы найти наиболее перспективные сегменты.  

*Данный проект выполнен путем написания запросов к базе данных в PostgreSQL.*

**Результаты:**
- Отфильтрованы выбросы и пропуски;
- Произведена категоризация по региону и времени размещения объявлений;
- Проанализирована сезонность объявлений;
- Сделаны общие выводы, даны рекомендации.
