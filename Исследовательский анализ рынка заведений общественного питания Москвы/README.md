# Исследовательский анализ рынка заведений общественного питания Москвы

[Ссылка на проект](https://github.com/daniel-melnikov/my-projects/blob/main/Исследовательский%20анализ%20рынка%20заведений%20общественного%20питания%20Москвы/Исследовательский%20анализ%20рынка%20заведений%20общественного%20питания%20Москвы.ipynb)

## Описание проекта
**Цель** исследовательского анализа - дать рекомендации заказчику по формату заведения для новой точки.  

**Задачи:**
- Ознакомиться с данными, провести их предобработку;
- Изучить категории заведений, их территориальное расположение, принадлежность к сети, вместимость, рейтинг, размер среднего чека;
- Проанализировать сетевые заведения Москвы.

## Используемые библиотеки:
- pandas
- matplotlib
- seaborn

## Результаты:
- Данные приведены к корректному формату;
- Проведена работа с пропусками: удаление, замена медианным значением;
- Объектные данные приведены к единому стилю;
- Удалены неявные дубликаты;
- Проанализированы заведения в разрезе категорий, количества посадочных мест, рейтинга;
- Даны рекомендации по формату и расположению для открытия нового заведения в Москве.
