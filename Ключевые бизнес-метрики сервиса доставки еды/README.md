# Ключевые бизнес-метрики сервиса доставки еды

[Ссылка на дашборд](https://github.com/daniel-melnikov/my-projects/blob/main/Ключевые%20бизнес-метрики%20сервиса%20доставки%20еды/Дашборд.png)

## Описание проекта:
Необходимо разработать обзорный дашборд по ключевым бизнес-метрикам сервиса доставки еды.

## Результаты:
- Рассчитаны DAU, динамика CR, Retention по дням и месяцам;
- Проведено сравнение AOV по месяцам;
- Выявлены топ-3 ресторана и топ-5 блюд по LTV.
