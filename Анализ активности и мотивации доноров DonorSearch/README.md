# Анализ активности и мотивации доноров DonorSearch

## Описание проекта
**Цель проекта** - выявить факторы, влияющие на активность доноров, разработать стратегии для их мотивации.  

*Данный проект выполнен путем написания запросов к базе данных в PostgreSQL*

## Результаты
- Выявлены наиболее активные доноры;
- Исследовано влияние системы бонусов на активность доноров;
- Рассмотрены каналы привлечения новых доноров;
- Проанализирована активность доноров, которые сдавали кровь повторно;
- Рассмотрена активность доноров в отношении к плану донаций компании.
