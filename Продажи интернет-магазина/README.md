# Дашборд по продажам интернет-магазина

[Ссылка на дашборд](https://github.com/daniel-melnikov/my-projects/blob/main/Продажи%20интернет-магазина/Дашборд%20по%20продажам%20интернет-магазина.png)

## Описание проекта:
Отдел продаж интернет-магазина попросил разработать обзорный дашборд по продажам товаров.

## Результаты:
- Посчитаны основные показатели: количество пользователей и покупателей, средний чек, конверсия;
- Построены графики объема товаров по категориям и объема и стоимость заказов;
- Построены графики динамики пользователей (в т.ч. расчитана доля активных) и затрат на рекламу по месяцам;
- Создана таблица с детализацией по заказам.
