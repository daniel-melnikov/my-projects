# Анализ покупок в видеоигре "Секреты Темнолесья"

[Ссылка на проект](https://github.com/daniel-melnikov/my-projects/blob/main/Анализ%20покупок%20в%20видеоигре%20Секреты%20Темнолесья/Анализ%20покупок%20в%20видеоигре%20Секреты%20Темнолесья.sql)

## Описание проекта
В данном проекте изучается влияние характеристик игровых персонажей и игроков, а также их активность внутри игры, на покупку внутриигровой валюты.

*Данный проект выполнен путем написания запросов к базе данных в PostgreSQL*

## Результаты:
- Исследованы доли платящих игроков;
- Рассмотрена статистика внутриигровых покупок;
- Выделены аномальные покупки;
- Рассмотрено количество приобретенных предметов в игре;
- Решена ad-hoc задача: зависит ли активность игроков от расы его персонажа.
