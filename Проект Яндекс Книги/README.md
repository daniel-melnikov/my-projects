# Проверка гипотезы в Python. Анализ результатов A/B-тестирования

[Ссылка на проект](https://github.com/daniel-melnikov/my-projects/blob/main/Проект%20Яндекс%20Книги/Проект%20Яндекс%20Книги.ipynb)

## Описание проекта
Первая часть проекта состоит из проверки гипотезы и проведения статистического теста, интерпретации его результатов.
Во второй части проекта необходимо провести проверку результатов другого A/B-теста.

## Используемые библиотеки:
- pandas
- scipy
- numpy
- statsmodels

## Результаты
В первой части проекта:
- Проведена оценка корректности данных;
- Проведена проверка гипотезы с помощью теста Манна-Уитни;
- Сформулирована аналитическая записка по итогам проекта;

Во второй части проекта:
- Произведена очистка данных, исключение пересечений;
- Проведен z-тест, интерпретированы результаты.
