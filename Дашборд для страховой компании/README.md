# Дашборд для страховой компании

[Ссылка на дашборд](https://github.com/daniel-melnikov/my-projects/blob/main/Дашборд%20для%20страховой%20компании/Дашборд%20для%20страховой%20компании.png)

## Описание проекта:
Необходимо разработать единый обзорный дашборд для руководителей всех подразделений компании.

## Результаты:
- Рассчитаны основные показатели: число клиентов, средний возраст, средний размер взноса;
- Определен суммарный доход по регионам, а также распределение среднего взноса по категориям;
- Рассчитано соотношение клиентов по полу и статусу курения;
- Построены визуализации распределения клиентов и среднего взноса по различным категориям;
- Определено количество клиентов по категориям;
- Вычислена корреляция категорий клиентов и их среднего взноса;
- Добавлена фильтрация для чартов "Распределение среднего взноса по категориям" и "Количество клиентов по категории".
