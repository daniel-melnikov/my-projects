# Анализ вакансий "Аналитик данных"

# Описание проекта
В данном проекте рассматриваются вакансии на позицию аналитика данных с целью выявления тенденций на рынке труда.  

*Данный проект выполнен путем написания запросов к базе данных в PostgreSQL и основан на исторических данных.*

# Результаты:
- Определены диапазоны заработных плат;
- Рассмотрено распределение вакансий по регионам и компаниям;
- Изучены типы занятости и рабочие графики, опыт соискателей;
- Исследованы hard и soft skills, требуемыt на позиции аналитика данных.
