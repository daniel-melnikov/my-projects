# Анализ рекламных кампаний для мобильного приложения

Ссылки на вкладки дашборда: [KPI](https://github.com/daniel-melnikov/my-projects/blob/main/Анализ%20рекламных%20кампаний%20для%20мобильного%20приложения/KPI.png), [Рекламный канал](https://github.com/daniel-melnikov/my-projects/blob/main/Анализ%20рекламных%20кампаний%20для%20мобильного%20приложения/Рекламный%20канал.png), [Тип рекламной кампании](https://github.com/daniel-melnikov/my-projects/blob/main/Анализ%20рекламных%20кампаний%20для%20мобильного%20приложения/Тип%20рекламной%20кампании.png), [Тип устройств](https://github.com/daniel-melnikov/my-projects/blob/main/Анализ%20рекламных%20кампаний%20для%20мобильного%20приложения/Тип%20устройств.png).

## Описание проекта:

Необходимо создать дашборд для оценки эффективности рекламных кампаний, активности пользователей и их конверсии в покупки внутри приложения.

## Результаты:

* На вкладке "KPI" рассчитаны ключевые бизнес-метрики;
* На вкладке "Тип рекламной кампании" изучена динамика ROI и соотношение доходов и расходов по типу рекламы;
* На вкладке "Рекламный канал" рассмотрена динамика ROI и сессий по каналам привлечения, а также средняя длительность одной сессии;
* На вкладке "Тип устройств" выявлено распределение сессий по девайсам, в т.ч. по месяцам и дням;
* Добавлена фильтрация по периоду, гранулярности, типу рекламы, рекламному каналу и типу устройства.
