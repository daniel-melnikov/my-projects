# TED конференции

[Ссылка на дашборд](https://github.com/daniel-melnikov/my-projects/blob/main/TED%20конференции/Дашборд%20TED%20конференции.png)

## Описание проекта:
Необходимо разработать информационный дашборд по прошедшим TED конференциям для компании, которая приобрела лицензию на проведение подобных конференций и готовит первое мероприятие.

## Результаты:
- Рассчитаны основные показатели: количество конференций, выступлений, спикеров и тем;
- Представлена детальная информация о конференциях и выступлениях;
- Выделены топ-10 смешных конференций и топ-20 тегов;
- Определен основной род деятельности спикеров.
- Добавлена фильтрация по странам, названию конференции, тегам, дате видеозаписи.
