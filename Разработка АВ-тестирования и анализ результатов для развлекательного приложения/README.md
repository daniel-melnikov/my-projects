# Разработка A/B-тестирования и анализ результатов

[Ссылка на проект](https://github.com/daniel-melnikov/my-projects/blob/main/Разработка%20АВ-тестирования%20и%20анализ%20результатов%20для%20развлекательного%20приложения/Разработка%20AB-тестирования%20и%20анализ%20результатов%20для%20развлекательного%20приложения.ipynb)

## Описание проекта
Команда разработчиков создала новый алгоритм рекомендаций для развлекательного приложения с короткими видео.  
**Задача:** рассчитать параметры и провести А/В-тест с целью проверки гипотезы о том, что новый алгоритм рекомендаций будет показывать более интересный контент.

## Используемые библиотеки:
- pandas
- matplotlib
- statsmodels
- math

## Результаты:
- Проанализированы количество регистраций пользователей, число просмотренных страниц в приложении;
- Рассчитана доля успешных первых сессий;
- Произведен расчет размера выборки, длительности теста;
- Произведена валидация корректного распределения пользователей по группам, в т.ч. по категориальным перенным, отсутствия пересечений в группах;
- Сформулированы нулевая и альтернативная гипотезы;
- Проанализирована ключевая метрика - доля успешных первых сессий;
- Проанализирована статистическая значимость изменения ключевой метрики;
- Сделаны выводы и даны рекомендации по итогам А/В-теста.
